## 6.27早会
- 明确以下注意点：

  - 每位成员需配置`.gitignore`文件防止提交不需要的文件

    参考配置如下：

    ```
    ### .gitignore ###
    # 忽略IDEA配置文件及生成文件
    .idea/
    .iml
    
    # 忽略gitignore文件
    .gitignore
    
    # 忽略编译文件
    target/
    *.class
    
    # 忽略本地的jdbc配置文件
    jdbc.properties
    
    # 忽略日志文件
    *.log
    ```

    

  - 为保证项目稳定性，项目主要配置文件（`pom.xml` /`spring-config.xml`/ `spring-mvc.xml` /`mybatis.xml`等）都由组长统一修改。
